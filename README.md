# DIGIY POS — Version Client

Ceci est la caisse officielle du commerçant utilisant le système DIGIY POS (0% commission).

## Contenu
- index.html : le logiciel complet de caisse
- Admin intégré (PIN : 9999)
- Mode caisse hors-ligne compatible mobile & tablette
- Paiements : Espèces, Wave, OM

## Notes
Ce répertoire est la version **client**.
Pour mettre à jour l'identité ou les produits, utiliser l'onglet **Admin** dans l'application.

## Mises à jour DIGIY
En cas de nouvelle version du moule DIGIY POS, remplacer le fichier index.html par celui fourni.
